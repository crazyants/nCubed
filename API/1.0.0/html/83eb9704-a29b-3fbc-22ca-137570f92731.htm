Remarks
Specification overload operators for create AND,OR or NOT specifications. Additionally overload AND and OR operators with the same sense of ( binary And and binary Or ). C# couldn’t overload the AND and OR operators directly since the framework doesn’t allow such craziness. But with overloading false and true operators this is posible. For explain this behavior please read http://msdn.microsoft.com/en-us/library/aa691312(VS.71).aspx
